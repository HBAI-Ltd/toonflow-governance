# Toonflow 社区治理章程

**v1.0 · 2026年3月24日 · HBAI-Ltd**

---

## 一、总则

本章程规定 Toonflow 社区理事会的构成、权利、义务，以及社区基金分红机制。

---

## 二、社区理事会

### 2.1 构成

社区理事会共 **15 席**：

| 席位类型 | 数量 | 产生方式 |
|---------|------|---------|
| 社区贡献者席 | 11 席 | 由全体有效贡献者选举产生 |
| HBAI 任命席 | 4 席 | 由 HBAI-Ltd 内部任命 |

### 2.2 贡献者席位竞选规则

**参选资格：**
- 年度贡献积分排名前 30 名内，方可参与竞选
- 历史累计积分 ≥ 200 分（防止短期刷分）

**选举规则：**
- 选举权：年度积分 ≥ 50 分的全体贡献者均可投票
- 每人最多投 11 票，票数前 11 名当选
- 选举结果公开公示，含各候选人得票数

**任期：**
- 任期 1 年，届满可申请连任
- 同一人连任不超过 **2 届**（防止席位固化）

**缺席处理：**
- 连续 2 次理事会会议无故缺席，席位由当年积分第 12 名自动递补
- 主动辞任需提前 30 天书面通知，由积分下一位递补

### 2.3 HBAI 任命席规则

- 由 HBAI-Ltd 内部任命，代表公司战略方向
- 任命名单每年公开公示
- HBAI 4 席持有**最终否决权**（用于保护项目核心方向和社区基本利益）
- 否决权使用须在公开会议记录中注明原因

## 三、理事会的权利与义务

### 3.1 权利

| 权利 | 说明 |
|------|------|
| **社区基金优先级投票权** | 每季度对基金使用方向进行投票，多数通过即执行 |
| **年度分红方案提案权** | 理事会可提出分红方案，交由 HBAI 最终确认 |
| **路线图知情权** | 提前知悉 Toonflow 产品路线图，可在公开前参与讨论 |
| **新功能内测优先权** | 优先获得新版本内测资格 |
| **官方代表资格** | 理事会成员可以"Toonflow 社区理事"身份参与对外活动 |

### 3.2 义务

| 义务 | 说明 |
|------|------|
| **参会义务** | 每季度理事会会议，无特殊情况须出席（线上/线下均可） |
| **公示义务** | 每次会议后 7 个工作日内，公开发布会议摘要和投票结果 |
| **中立义务** | 投票时须以社区整体利益为优先，不得代表个人或商业利益进行利益输送 |
| **保密义务** | 路线图等未公开信息，在官方发布前不得对外泄露 |
| **行为规范** | 遵守 Toonflow 社区行为准则，理事会成员的公开言行代表社区形象 |

### 3.3 关于分红

- **理事会席位本身不附带任何分红资格**——治理权与经济权完全分离
- 理事会成员若同时拥有个人贡献积分，**以普通开发者身份**参与社区基金分红池，积分不因席位身份加权或减权
- HBAI 任命席成员同等适用上述规则

### 4.1 积分体系状态

> **状态：进行中，待进一步讨论**

> 详细积分规则以最终公示版本为准。理事会鼓励社区在此期间积极提出意见，可通过 GitHub Issue 或私信反馈。

## 五、社区基金分红机制

### 5.1 资金来源

全部商业授权收入的 **50%** 进入社区基金，季度划拨，季度公示。

### 5.2 分红资格

- **参与门槛：** 年度积分 ≥ 50 分
- **分配方式：** 按各人年度积分占全体有效贡献者积分总和的比例分配
- **分配范围：** 所有满足门槛的贡献者，含理事会成员（以个人积分参与，席位不加权）

### 5.3 分红计算示例

> 假设某年度：
> - 社区基金分红池：20 万元
> - 满足门槛（≥50分）的贡献者：40 人
> - 全体有效贡献者年度积分总和：10,000 分
> - 某贡献者年度积分：500 分
>
> **该贡献者分红 = 20万 × (500 / 10,000) = 1万元**

### 5.4 透明机制

| 频率 | 内容 |
|------|------|
| 每季度 | 公布授权收入总额 + 社区基金划拨金额 |
| 每半年 | 公示基金使用明细 |
| 每年度 | 公示分红名单（含各人积分、分红金额，本人可选择匿名） |

## 六、过渡安排

在第一届正式选举完成前（预计6个月内）：

- 授权决策权暂由创始理事会（创始委任制）代为行使
- 法人主体及法律责任由 HBAI-Ltd 暂时代行
- 授权合同以 HBAI-Ltd 为主体签署
- 社区见证人全程参与所有授权决策讨论
- 选举完成后，授权权限正式移交理事会，法人主体变更条件由理事会投票决定

## 七、章程修订

- 本章程修订需经理事会投票通过（贡献者席 11 席中 ≥ 7 席同意）+ HBAI 最终确认
- 修订内容须提前 14 天公示草案，接受社区反馈
- 每次修订记录版本号和修订说明，历史版本存档公开
